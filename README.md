## 🚀 What I'm working on

### [llm-self-evolve](https://github.com/TakuroidX/llm-self-evolve) ⭐
A framework for **LLM-powered autonomous code improvement** with multi-layer safety.
Claude Sonnet 4 observes a production system, identifies failure patterns,
generates code (multi-file, 3-layer), validates statistically, and creates PRs
for human review. Production-tested over 15 months.

> *Sprint 7 + Phase 3 = the loop closes itself. Human only approves merges.*

### Cryptocurrency Trading Bot (private, 15 months in production)
The reference implementation domain for `llm-self-evolve`.
24/7 simulation, multi-layer filtering, LSTM + XGBoost + technical analysis fusion.

### [survivor-brain-pi](https://github.com/TakuroidX/survivor-brain-pi)
Open-source non-invasive head impact monitor for combat sports.
Hardware × ML × athlete safety.
